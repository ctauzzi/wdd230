Jennifiori Site Plan


Site Purpose

The purpose of this website is to have a site plan for the website called Jennifiori, a website for a floral business.
It will show the target audience, the color scheme and the typography of the website.


Domain Name

jennifiori.com



Target Audience

The audience for this website would be anyone desiring either specific floral arrangment requests or also those who are seeking wedding arrangements, bouqets, and any other floral requirements for weddings.



Color Scheme

Main

The main background color will be #C1E1C1 (pastel green).


Headings and paragraphs

Headings and paragraphs will be #040403 (black).


Navigation

The navigation bar will have the background color of #E4E0D7 (alabaster) and with a text color of #040403 (black).


Footer

The footer will have a background color of #E4E0D7 (alabaster) and a text color of #040403 (black).

Typography

Headings

Noto Sans, Regular 400


Paragraphs and all regular text

Roboto Condensed, Light 300
